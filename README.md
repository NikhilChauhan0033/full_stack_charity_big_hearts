Big Heart ( Charity Platform ) Web App - FULLSTACK

Languages used - Djnago, Djnago Rest Framework, React, MUI, SQLite

Developed a full-stack charity donation platform using Django, Django REST Framework, and React, enabling users to browse campaigns, make donations, and track their activity with a clean and responsive UI built using Tailwind CSS and Material UI. Implemented secure user authentication and authorization using JWT tokens, built a scalable REST API, and integrated a custom admin panel allowing only authorized admins to add, update, and manage donation campaigns and user activity.
